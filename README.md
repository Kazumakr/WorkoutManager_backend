## Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Kazumakr/
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Enter parameters for MySQL connection in `.env`.
   ```shell
   PORT=3001
    USERNAME=
    PASSWORD=
    DATABASE=
    HOST=
   ```
4. start
   ```sh
   npm start
   ```
